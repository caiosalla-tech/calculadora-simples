# calculadora-simples
O projeto consiste em uma calculadora desenvolvida com HTML5 e CSS3, com foco em organização estrutural, clareza visual e escalabilidade. A estrutura foi construída de forma semântica, separando container principal, display e área de botões, garantindo hierarquia e legibilidade do código.

O layout dos botões foi implementado com CSS Grid, organizado em uma matriz de quatro colunas com controle de espaçamento por meio de gap. O uso de grid-column: span no botão de igual demonstra domínio de variações estruturais dentro da mesma grade. A centralização do componente na tela foi realizada com Flexbox, assegurando alinhamento vertical e horizontal consistente.

A diferenciação visual dos botões foi feita por classes específicas conforme responsabilidade (números, operadores, limpar e resultado), evidenciando preocupação com organização, manutenção e futura integração com JavaScript. Também foi aplicado reset básico com box-sizing: border-box, garantindo previsibilidade no layout.

A responsividade foi considerada por meio da meta tag viewport, permitindo adaptação adequada em dispositivos móveis.

Embora esta versão seja apenas visual, a estrutura foi pensada para expansão com JavaScript modularizado, permitindo futura implementação da lógica de cálculo sem necessidade de refatoração estrutural. O projeto demonstra domínio de fundamentos essenciais de front-end: semântica, Grid, Flexbox, organização de CSS e visão de escalabilidade.
